Cloud Computing Tasks 1-4

Overview

This repository contains the implementation of four cloud computing tasks developed using Python and Visual Studio Code. These tasks demonstrate basic cloud computing concepts such as data processing, storage management, automation, and simple cloud-based applications.

Tasks Completed

Task 1: Data Redundancy Removal

Objective: Remove duplicate records from a dataset and store only unique values.

Features:

Reads input data.

Identifies duplicate entries.

Removes redundant records.

Displays cleaned data.


Technology Used:

Python

Visual Studio Code



---

Task 2: Cloud Storage Simulation

Objective: Simulate cloud storage operations such as uploading, viewing, and deleting files.

Features:

Upload files.

Display stored files.

Delete files.

Manage storage records.


Technology Used:

Python

File Handling



---

Task 3: Resource Monitoring

Objective: Monitor system resources and display CPU and memory usage.

Features:

CPU utilization monitoring.

Memory usage monitoring.

Real-time statistics display.


Technology Used:

Python

psutil Library



---

Task 4: Subtitle Chatbot

Objective: Develop a simple chatbot that generates or manages subtitles for video content.

Features:

Accepts subtitle input.

Displays subtitle responses.

Basic chatbot interaction.


Technology Used:

Python

Natural Language Processing Basics

Installation

1. Clone the repository:



git clone <repository-url>

2. Open the project in Visual Studio Code.


3. Install required dependencies:



pip install -r requirements.txt

4. Run the desired task:



python task1.py
python task2.py
python task3.py
python task4.py
